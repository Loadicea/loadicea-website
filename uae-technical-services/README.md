# Loadicea Heating Solutions Trading

A professional B2B website for Solar Repair, AMC, and Facility Maintenance services.

## How to View
Since this project uses pure static HTML/CSS (no build tools required):
1.  Navigate to the project folder.
2.  Double-click `index.html` to open it in your web browser.

## Project Structure
*   `index.html`: Homepage (Projects portfolio, Service overview).
*   `solar-maintenance.html`: Repair, AMC, and Inspection details.
*   `building-maintenance.html`: Electrical, Plumbing, Painting, AC scope.
*   `contact.html`: Inquiry form and contact details.
*   `css/`: Contains the Design System (variables, components, layout).
